# TestnetForFabricDev
This is a testnet for Hyperldedger Fabric developer based docker.

## Getting Started

**Start the network:**
```
$ ./testnet.sh up
```

**Stop the network:**
```
$ ./testnet.sh down
```

**Get in the cli.testnet.com container to operator the network:**
```
$ docker exec -it cli.testnet.com bash
```
